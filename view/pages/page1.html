开始

在页面的开头通过 <script> 引入View类库。下载

示例：

      
      <!DOCTYPE html>
      <html>
      
      <head>
        <title>viewjs</title>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <script type="text/javascript" src="release/view.js"></script>
      </head>
      
      <body>

        {{text}}
        
      </body>
      <script type="text/javascript">
     
      var app = new View({
        view: "body",
        model: {
          text: "Hello, world!"
        }
      });
     
      </script>
      </html>

兼容性

由于采用了最新的HTML和es6的某些标准，不支持低版本的浏览器，如IE10以下本版，但是支持最新主流的桌面浏览器，和移动浏览器
