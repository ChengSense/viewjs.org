插值

插值用于把绑定相应变量的内容以文本的格式输出到视图和其他的框架一样用双大括号来表示。

语法 

    
    {{表达式}}

示例 

    
    <span id="msg">Message: {{msg}}</span>
    
    
    
    new View({
      view: "span#msg",
      model: {
        msg: "输入点什么?"
      }
    });
    
    
    
    输入点什么?

条件

View里的条件指令，相对于其他的js框架的条件的指令更容易理解（View的诞生其实和AngularJS糟糕难用的ng-if指令，多少有点关系。 在某个风和日丽的上午或者下午，当然现在记不太清了，当用到ng-if指令实现某个效果的时候，被坑了一些时间，现在想想当时心里估计有1w个曹尼玛奔腾而过， 当时就在想设计ng-if指令的人脑子是不是坏掉了，已于常人难以理解。于是对其好感度急剧下降，在想有没有一套更简单的前端框架，百度，谷歌然而并没有找到； 随后在某个晚上开始撸起代码）。

语法 

    
    @when(布尔表达式){
      业务逻辑
    }

示例

    
    <div id="msg">
      @when(show){
        <span>Message: {{msg}}</span>
      }
    </div>
    
    
    
    new View({
      view: "div#msg",
      model: {
        msg: "输入点什么?",
        show : true
      }
    });
    
    
    
    输入点什么?

条件分支

@when 指令要配合 .when 指令一起使用,可以实现 if else 效果

语法

    
    @when(布尔表达式){
      业务逻辑
    }
    .when(布尔表达式){
      业务逻辑
    }
    .when{
      业务逻辑
    }

示例

    
    <div id="msg">
      @when(show){
        <span>Message: {{ msg }}</span>
      }
      .when{
        hello wold ！
      }
    </div>
    
    
    
    new View({
      view: "div#msg",
      model: {
        msg: "输入点什么?",
        show : false
      }
    });
    
    
    
    hello wold ！

循环

语法

    
    @each(变量：主键(可以省略)：数组||对象){
      业务逻辑
    }

示例

    
    <ul id="ul">
      @each(li:ul){
        <li>
          {{li.text}}
        </li>
      }
    </ul>
    
    
    
    new View({
      view: "ul#ul",
      model: {
        ul: [{
          text :"View.js"
        },{
          text :"Vue.js"
        },{
          text :"React.js"
        }]
      }
    });
    
    
    
      
    
        
  • View.js
    • 
        
  • Vue.js
    • 
        
  • React.js
    •

注意点：循环表达式的参数 “变量：数组||对象”，如需要对象的主键这可以写成 “变量：主键(可以省略)：数组||对象” 主键的定义不是必须的，不需要的话可以省略

循环属性

循环属性 each请参考 @each指令用法并无不同

语法

    
    <div each="item:key:list"></ldiv>

示例

    
    <ul id="ul">
      <li each="text:index:ul">
        {{ index + 1 }}. {{ text }}
      </li>
    </ul>
    
    
    
    new View({
      view: "ul#ul",
      model: {
        ul: ["View.js","Vue.js","React.js"]
      }
    });
    
    
    
    
    
        
  • 1.View.js
    • 
        
  • 2.Vue.js

  • 3.React.js

多层嵌套

多层嵌套，通过复杂的指令组合，完成想你所想效果

示例

    
    <ul id="ul"each="text:index:ul">
      @each(item:ul){
        <li>
        @when(index==1}{
          {{index + "." + text }}
        }
        .when{
          {{index}}. hello wold ！
        }
        </li>
      }
    </ul>
    
    
    
    new View({
      view: "ul#ul",
      model: {
        ul: ["View.js","Vue.js"]
      }
    });

用户输入

通过value={{ 变量 }}属性，就能简单的完成数据的双向绑定

示例

    
    <div id="input">
      <div> {{text}} </div>
      <input value="{{text}}"/>
    </div>
    
    
    
    new View({
      view: "div#input",
      model: {
        text:"输入点什么?"
      }
    });

SELECT元素

实例

    
    <section value="{{msg}}">
      <option value="1">苹果</option>
      <option value="2">梨</option>
    </section>
    
    
    
    new View({
      view: "section",
      model: {
        msg:""
      }
    })

注意点： 如果model.msg有默认值的话，会成为SECTION的默认选项，如果option设定了selected="ture"则model.msg会被设置成默认值

TEXTAREA元素

示例

    
    <textarea value="{{msg}}">
      {{msg}}
    </textarea>
    
    
    
    new View({
      view: "section",
      model: {
        msg:""
      }
    })

注意点： 标签里层也要写 value 同样的变量，不然内容会不显示
